## 🎨 CUSTOMIZATION EXAMPLES

### Change Earth rotation speed (edit config.js)
```javascript
// Line ~15
EARTH: {
    ROTATION_SPEED: 0.001,  // Default (1-second rotation per 1000 frames)
    // ROTATION_SPEED: 0.002,  // 2x faster
    // ROTATION_SPEED: 0.0005, // 2x slower
}
```

### Change quality for performance (edit config.js)
```javascript
// Pre-configured presets in earth-vr.js:
applyPreset('PERFORMANCE');  // For slow devices
applyPreset('BALANCED');     // Default (this was used)
applyPreset('QUALITY');      // For powerful machines
```

- Templates provided for each
- Just copy-paste and customize
